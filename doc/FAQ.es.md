<details>
  <summary><b>El archivo de texto con la configuración de teclas no existe</b></summary>
  <br/>
  Si el archivo no existe, hay que arrancar el juego con el mod instalado al menos una vez.
</details>
<details>
  <summary><b>Los botones de mando por defecto no corresponden a los botones superiores del mando</b></summary>
  <br/>
  Los identificadores de botones de los mandos pueden variar de un modelo a otro. Puedes usar la herramienta de configuración de mandos de tu sistema operativo, o comprobadores de mandos online, para verificar el identificador que usan los botones de tu mando.
</details>
<details>
  <summary><b>El botón del mando no funciona a pesar de que el identificador es correcto</b></summary>
  <br/>
  A veces, sobre todo cuando se tienen conectados varios mandos USB o se cambia de enchufe un mando específico, el sistema operativo le asigna un identificador diferente al mando en si. Hay que asegurarse de que la configuración es para el mando correcto, o bien usar los identificadores universales (<i>any gamepad</i>) que hay en la referencia de Unity.
</details>
<details>
  <summary><b>La Rueda de Plegarias no funciona en algunas situaciones, como el combate contra Isidora o  el demake</b></summary>
  <br/>
  Esto es correcto, en situaciones donde el juego base no permite usar el inventario, la Rueda de Plegarias se desactiva.
</details>
<details>
  <summary><b>Con algunas plegarias, si están activas, al cambiar de plegaria se desactivan ¿Es normal?</b></summary>
  <br/>
  Si, este es el funcionamiento normal del juego.
</details>
<details>
  <summary><b>¿Es esta modificación compatible con otras?</b></summary>
  <br/>
  En principio deberia serlo, la Rueda de Plegarias se añade al Penitente del juego base, y usa el inventario normal para activar las Plegarias.<br>
  Podría darse el caso de que si una modificación cambia muy drasticamente alguno de estos sistemas, surjan incompatibilidades.
</details>
<details>
  <summary><b>¿Se pueden añadir nuevas plegarias?</b></summary>
  <br/>
  En principio si, la rueda de plegarias usa el inventario estandard. Si las nuevas plegarias se añaden al sistema normal, deberían funcionar sin problema.
</details>
