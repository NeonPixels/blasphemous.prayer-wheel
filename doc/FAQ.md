<details>
  <summary><b>The configuration text file for the keybindings does not exist</b></summary>
  <br/>
  If the text file doesn't exist, run the game with the mod installed at least once.
</details>
<details>
  <summary><b>The default gamepad buttons do not match the shoulder buttons</b></summary>
  <br/>
  Gamepad keys can vary from gamepad model to gamepad model, you can check your operating system configuration tool or use an online gamepad tester to determine the specific identifiers for your hardware.
</details>
<details>
  <summary><b>The gamepad button is not working despite being configured to the correct button number</b></summary>
  <br/>
  Sometimes, specially when having multiple USB gamepads connected or when connecting a gamepad to a different USB port, the operating system can assign a different identifier to the gamepad, make sure your configured binding matches the gamepad, or use the <i>"any gamepad"</i> bindings found in the Unity reference.
</details>
<details>
  <summary><b>The Prayer Wheel doesn't work in specific areas, like the Isidora boss fight or during the demake</b></summary>
  <br/>
  This is correct. In situations where the base game doesn't allow use of the inventory, the Prayer Wheel will also be disabled.
</details>
<details>
  <summary><b>Some Prayer effects seem to dissipate when switching to a different prayer</b></summary>
  <br/>
  This is the base game behaviour.
</details>
<details>
  <summary><b>Is this mod compatible with other mods?</b></summary>
  <br/>
  It should be, the Prayer Wheel itself is attached to the base game Penitent, and makes use of the base game inventory system to witch prayers.<br>
  Nevertheless, if a mod alters those base game elements, there might be compatibility issues.
</details>
<details>
  <summary><b>Can mods add new Prayers?</b></summary>
  <br/>
  It should be possible, as long as those Prayers make use of the base game inventory system.
</details>
